# AI Sales Agent

An intelligent **AI-powered sales agent** that automates product recommendation, negotiation, order handling, and follow-up using a multi-agent workflow. The system leverages **LangGraph**, **FAISS vector search**, and **AI models** to enhance sales efficiency across multiple platforms like WhatsApp, Instagram, and web chat.

---

## 🚀 Features

- **Intent Detection** – Classifies user messages into intents using AI.
- **Product Recommendation** – Suggests products based on user queries and preferences.
- **Negotiation Agent** – Handles price negotiations intelligently.
- **Order Handling** – Processes orders and tracks them automatically.
- **Follow-up Automation** – Sends reminders, discounts, and notifications to customers.
- **Vector Store Search** – Fast semantic search across product catalog using FAISS embeddings.
- **Multi-Channel Support** – Integrates with WhatsApp, Instagram, and Web UI.
- **Conversation Memory** – Remembers customer history for personalized interactions.
- **Streamlit Chat UI** – Simple frontend for testing and demonstration.
